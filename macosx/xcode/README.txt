Crrcsim installation instructions on macosx :
Double-click .dmg.tgz file to mount disk image. A new disk will appear containing the icon of your new crrcsim application Ñ simply drag this icon to whatever folder or disk you'd like to store it in.
You may then "eject" and disccard this disk image.
